SDE Learning Trail
SDE Learning Trail is a simple, easy to use web application to help users learn, interact and engage with SDE4.
Systems as Trails
SDE Learning Trail describes SDE4â€™s six different building systems as trails for occupants and building visitors.
Stations
Each trail is composed of a number of physical "stations", placed at different areas in SDE4.
Digitalisation of Stations
QR codes and a web application are used to connect stations with information and visualizations explaining each building system.
Comfort Feedback
Users are prompted to provide comfort feedback while going through each trail. This helps the building team understand how occupants feel in the new building.
Curation of Digital Content
This new human-building interaction framework helps occupants and visitors to easily engage with SDE4 to appreciate its design, construction, and sustainability features.
